## What's included

- CheckApp Mac application for Fennoa receipt entry.
- CheckApp login with email code. Google users use the same Google email and receive a one-time code.
- Receipt folder selection in a CheckApp window.
- Data processing acknowledgement before receipt recognition.
- Fennoa login and receipt review in the Fennoa browser window.
- Local receipt processing flow with processed/failed folders.
- Receipt recognition through the CheckApp backend proxy.
- App icon embedded in the macOS application bundle.

## Notes

- CheckApp is an early release for controlled launch testing.
- Users should always review and correct recognized data in Fennoa before using it for bookkeeping, tax reporting or payments.
- The app is ad-hoc signed, not Apple-notarized.
