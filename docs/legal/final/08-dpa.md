# HENKILÖTIETOJEN KÄSITTELYSOPIMUS

Päivitetty: 7.7.2026

## 1. Osapuolet ja roolit

Asiakas toimii rekisterinpitäjänä niille henkilötiedoille ja kuittiaineistoille, joita se käsittelee CheckAppin avulla.

ART-HAUS toimii henkilötietojen käsittelijänä siltä osin kuin se käsittelee henkilötietoja CheckApp-palvelun tuottamiseksi asiakkaan lukuun.

## 2. Käsittelyn tarkoitus

Käsittelyn tarkoitus on CheckApp-palvelun tarjoaminen, kuittien lataaminen Fennoaan, kuittitietojen tunnistaminen, tilauksen ja käyttökiintiön tarkistaminen sekä tekninen tuki.

## 3. Kuittikuvien käsittely

CheckApp käsittelee asiakkaan kuittikuvaa väliaikaisesti backend-välityspalvelussa ainoastaan tietojen tunnistamisen toteuttamiseksi ja OpenAI API -palveluun välittämiseksi. CheckAppin backend-palvelua ei ole tarkoitettu kuittikuvien, Base64-sisällön, OpenAI:n raakavastausten tai tunnistettujen kirjanpitokenttien pysyvään säilyttämiseen.

Käsittelyn jälkeen Mac-sovellus käyttää tunnistettuja tietoja Fennoan kenttien täyttämiseen paikallisen selainautomaation avulla käyttäjän Fennoa-istunnossa.

## 4. Fennoa-tunnukset

Fennoa-salasanaa tai Fennoan 2FA-koodia ei lähetetä CheckAppin backend-palveluun eikä OpenAI API -palveluun. Käyttäjä kirjautuu Fennoaan Fennoan omassa kirjautumisnäkymässä.

## 5. Alikäsittelijät

ART-HAUS voi käyttää alikäsittelijöinä esimerkiksi OpenAI:ta, Stripea, Supabasea, Verceliä, Googlea ja GitHubia siinä määrin kuin palvelun tarjoaminen edellyttää.

## 6. Tietoturvapoikkeamat

ART-HAUS arvioi mahdolliset tietoturvapoikkeamat tapauskohtaisesti ja noudattaa soveltuvaa GDPR:n 72 tunnin ilmoitusmenettelyä silloin, kun se on tarpeen.
